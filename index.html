DigitalGalaxy

Description

Digital Galaxy is the electronic devices store responsive, full-stack app that allows customers buy the products from the store. All users, orders, products and favourite products of users are stored in MongoDB. Cart and favourites when user is not logged in, are stored in Redux with local storage.

Features

  • - account setup
  • - editing user's account
  • - adding to cart
  • - adding to favourites
  • - making orders
  • - paying for orders (PayPal demo)
  • - marking orders as payed and delivered
  • - products quantity management
  • - adding new products

Technologies

  • JavaScript library: React
  • Styling: React Bootstrap
  • UI State Management: Redux
  • Backend: MongoDB + Express.js + Node.js
  • Routing: React Router

TripTracker

Description

TripTracker is the web app that allows users to tap on the map from Leaflet library and save their trips. Markers appear on the map and all the trips are listed with the name of the city, region, country, date and description. Moreover, all countries and continents visited during the trips are on a separate list. User can click the country on the list to get some basic information like e.g. flag, coat of arms, population, currency, timezones.

Features

  • - account setup
  • - adding and removing trips
  • - placing markers of the trips on the map
  • - listing all places (city or region) visited
  • - listing all countries visited
  • - listing all continents visited
  • - getting basic country's information

Technologies

  • JavaScript library: React
  • Styling: Tailwind
  • UI State Management: Redux
  • Backend: MongoDB + Express.js + Node.js
  • Routing: React Router

BudgetMaster

Description

BudgetMaster is the web app that allows users to track their budget in terms on monthly or yearly basis. Each user registering to the service, gives information about their income and savings goal of the month (it is possible to change it anytime). User can add expenses to every single month and there is possibility of adding an expense to every month to reduce user's work. Expenses are categorized and their total values are displayed on a pie chart. Yearly stats are also available. Everything with the comments. There are also progress bars that shows if goal is crossed and if how many % of income have been spent. User can change currency with live exchange rate provided by Frankfurter API. Currencies provided are USD, EUR, GBP, CHF and PLN.

Features

  • - account setup
  • - deleting account
  • - setting own income and savings goal of user
  • - adding an expense to a month or to every month simultaneously
  • - adding desctiption and choosing category for an expense
  • - pie chart with categories of expenses summed up
  • - progress bar of expenses and goal of savings
  • - exchanging currency with live exchange rate

Technologies

  • JavaScript library: React
  • Styling: Styled Components
  • UI State Management: Context API + Reducer
  • Backend: API deployed on Render
  • Routing: React Router
