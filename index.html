    DigitalGalaxy

    Description

    Digital Galaxy is the electronic devices store responsive, full-stack app that allows customers buy the products from the store. All users, orders, products and favourite products of users are stored in MongoDB. Cart and favourites when user is not logged in, are stored in Redux with local storage.

    Features

    • - account setup
    • - editing user's account
    • - adding to cart
    • - adding to favourites
    • - making orders
    • - paying for orders (PayPal demo)
    • - marking orders as payed and delivered
    • - products quantity management
    • - adding new products

    Technologies

    • Styling: React Bootstrap
    • Routing: React Router
    • UI State Management: Redux
    • Backend: MongoDB + Express.js + Node.js
