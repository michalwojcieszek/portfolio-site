  • Mern app

    TripTracker

    Description

    TripTracker is the web app that allows users to tap on the map from Leaflet library and save their trips. Markers appear on the map and all the trips are listed with the name of the city, region, country, date and description. Moreover, all countries and continents visited during the trips are on a separate list. User can click the country on the list to get some basic information like e.g. flag, coat of arms, population, currency, timezones.

    Features

    • - account setup
    • - adding and removing trips
    • - placing markers of the trips on the map
    • - listing all places (city or region) visited
    • - listing all countries visited
    • - listing all continents visited
    • - getting basic country's information

    Technologies

    • JavaScript library: React
    • Styling: Tailwind
    • UI State Management: Redux
    • Backend: MongoDB + Express.js + Node.js
    • Routing: React Router
